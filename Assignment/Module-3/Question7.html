How to install React Js on Windows, linux Operating System? How to install NPM and How to check version of NPM?

To install React.js, you first need to install Node.js and npm (Node Package Manager), as they are prerequisites.
1. Install Node.js and npm
For Windows:
1. Go to the Node.js official website.
2.Download the LTS (Long Term Support) version.
3.Run the installer and follow the prompts to complete the installation.
4.During the installation, make sure to check the option to install npm.

For Linux:
1. Open a terminal.
2. Use the following commands to install Node.js and npm. The specific commands depend on your Linux distribution.
3. sudo apt install nodejs npm

by command npm -v we can check the version of npm.

Install React.js using Create React App:
1.open terminal or command prompt
2.type npx create-react-app my-app command.
